 
# Panda3D 赛车游戏

使用 Panda3D 引擎开发的简易赛车游戏。

## 🎮 游戏特性

- **红色赛车**：由车身、车顶和4个轮子组成
- **键盘控制**：WASD 或 方向键控制赛车
- **相机跟随**：相机自动跟随赛车移动
- **速度显示**：实时显示当前速度
- **赛道标记**：红蓝色标记物作为赛道边界

## 🎯 控制方式

| 按键 | 功能 |
|------|------|
| W / ↑ | 加速 |
| S / ↓ | 刹车/倒车 |
| A / ← | 左转 |
| D / → | 右转 |
| ESC | 退出游戏 |

## 📦 安装依赖

```bash
pip install -r requirements.txt
```

## 🚀 运行游戏

```bash
python main.py
```

## ⚙️ 游戏参数

| 参数 | 值 |
|------|-----|
| 最大速度 | 100 km/h |
| 加速度 | 50 km/h² |
| 减速度 | 30 km/h² |
| 转向速度 | 150 °/s |
| 相机距离 | 10 单位 |
| 相机高度 | 5 单位 |

## 🎨 代码结构

### 主要方法

| 方法 | 说明 |
|------|------|
| `__init__()` | 初始化游戏 |
| `setup_scene()` | 设置场景（地面、赛道） |
| `setup_car()` | 设置赛车模型 |
| `setup_camera()` | 设置相机 |
| `setup_inputs()` | 设置键盘输入 |
| `setup_ui()` | 设置UI界面 |
| `update()` | 游戏主循环 |
| `update_car_transform()` | 更新赛车位置 |
| `update_camera()` | 更新相机位置 |
